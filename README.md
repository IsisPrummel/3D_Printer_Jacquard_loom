# 3D_Printer_Jacquard_loom
This github contains all components to recreate the computational Jacquard weaving loom.
The toolkit consists of a loom add-on created for a Creality Ender 3 pro 3D printer and a conversion tool to convert your designed weave draft (in AdaCAD) to executable printer instructions (G-code).

This github contains all parts and instructions to recreate the loom. It contains a part list (including sizing and estimated price), assembly instructions and the conversion tool file. It also contains all parts that can be fabricated on the 3D printer and laser cutter, the rest of the parts can be recreated with basic workshop tools following the assembly instructions.
